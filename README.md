# ALPHANUMERIC MERGE SORT

## RUN 

py alphanumeric_merge_sort.py

The script will run with the following data and result:

- ["ABC1", "ABC001", "01", "0", "Zzz", "zzz"] = ['0', '01', 'ABC001', 'ABC1', 'Zzz', 'zzz']
- ["4","3","2","1","0"]                       = ['0', '1', '2', '3', '4']
- [4,3,2,1,0]                                 = [0, 1, 2, 3, 4]
